This is a mod to add robots in minetest. There is actualy only one robot: the brigadier, a simple soldier robot.

[ STRUCTURE OF THE PROJECT ]

This mod follows the minetest standard mod structure with some specifications added.
Each robot project is represented by a .blend file in blender directory. attached scripts are included in. It is possible to modify a robot just from the .blend file. (according to have blender of course.)

[ WHY IS THERE BLENDER FILES ? ]

This mod is distributed under General Public License, so I give the source of this mod with the mod.
The .blend file have been used to create the robots's meshs and scripts. It is possible to use and modify the present mod without blender or gimp.
